## to get started (commands asume mac or linux):
```
git clone https://github.com/ZKemstedt/devops20_python_slutuppgift.git zkemstedt
cd zkemstedt
pipenv install
```

## to run the different modules
```
pipenv run client
pipenv run server
pipenv run test
pipenv run report
```
